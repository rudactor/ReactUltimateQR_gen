# QR Generator + UI (React + Vite + TailwindCSS)

## Overview

This project is a customizable QR code generator built with **React**, **Vite**, and **TailwindCSS**.  
It focuses on a clean UI, fast performance, and flexible customization options.

Users can generate QR codes and adjust visual properties such as colors, size, and style, with instant preview and easy export for further use.

## Tech Stack

- Javascript
- React
- Vite
- TailwindCSS

## Features

- QR code generation with live preview
- Visual customization (colors, size, style)
- Modern, responsive UI
- Fast development and build with Vite
